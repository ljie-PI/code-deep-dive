# Claude Code 深度解析

> 基于 Claude Code v2.1.888 逆向工程代码库的系统性技术文档。

## 这是什么

本系列文档是对 Anthropic 官方 Claude Code CLI 工具的深度技术解析，共 15 章，覆盖从启动流程到设计哲学的完整技术栈。

**信息来源**：
- 逆向工程/反编译的源代码（~1341 个 tsc 类型错误来自反编译，不影响运行时行为）
- 源码级 grep 验证的环境变量与配置
- Mermaid 图表均为基于代码分析的原创绘制

**适合谁**：
- 想理解 AI Agent 工程实践的开发者
- 计划 Fork 或二次开发 Claude Code 的团队
- 对 LLM 应用架构感兴趣的架构师
